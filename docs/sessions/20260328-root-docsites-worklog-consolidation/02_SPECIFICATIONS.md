# Specifications

- Maintain session documentation in `docs/sessions/<session>` only.
- Canonical session artifacts are: `00_SESSION_OVERVIEW.md`, `01_RESEARCH.md`, `02_SPECIFICATIONS.md`, `03_DAG_WBS.md`, `04_IMPLEMENTATION_STRATEGY.md`, `05_KNOWN_ISSUES.md`, `06_TESTING_STRATEGY.md`.
- Legacy `index.md`/`WORKLOG.md` entries are sources to be archived or merged, not treated as canonical runtime work log.
